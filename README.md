# inArtists Pay Portal Generator

A comprehensive React application for generating client payment portals with live preview and CSV export functionality.

## Features

- **Live Portal Preview** - Real-time preview of client dashboard
- **CSV Export** - Export channel data, activity, and comprehensive reports
- **Customizable Branding** - Full control over client portal appearance
- **Data Management** - Easy input management for all portal metrics
- **Responsive Design** - Works on desktop and mobile devices

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open http://localhost:5173

## Export Functionality

- **Channel Data Export** - Export platform earnings data
- **Activity Timeline Export** - Export recent activity items
- **Comprehensive Export** - Export all portal data in structured format
- **Client Portal CSV** - End clients can export their data directly
